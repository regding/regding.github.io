关于本Blog 
---

### 为什么建本Blog
多半是出于自私的个人目的，为了有一个可以记录自己学习笔记，甚至是记录一些配置的自留地。

### 为什么选择在Github Pages
为了不那么折腾，只需要一个简洁、稳定的Blog站点。

### 用到的资源
* [Jekyll][]	本地环境以及Github上用于将文本文件解析成HTML静态页面的引擎
* [Emerald theme][]	一个给Jekyll搭建Blog的主题，本Blog就是基于此主题修改而成
* [Jekyll-bootstrap][]	一个可以帮助小白用户快速搭建Jekyll站点的工具集合或者框架
* [Kiko-plus][]	另一个Jekyll的主题，使用了Rakefile的部分功能和上述资源的Rakefile做了整合
